# Notes

#### Using wkhtml2pdf

Command line examples

```bash
wkhtmltopdf -q http://google.com google.pdf
```

#### ImageMagick

Most ideal command for preview image creation
```bash
convert -density 150 tax.pdf[0] -quality 70 -flatten tax.png
```
